Belajar Spacing

margin

Pada umumnya, Anda akan menemui penggunaan atribut type ini saat menentukan jenis bahasa atau sintaksis CSS yang digunakan dalam elemen tag style. Misalnya, jika Anda ingin menggunakan Tailwind CSS, Anda dapat mengatur atribut type sebagai text/tailwindcss untuk memberi tahu browser atau mesin pengurai bahwa konten di dalam elemen tag style adalah kode Tailwind CSS.


padding

margin ditailwind bisa diberikan auto dan negatif, namun untuk padding ditailwind tidak bisa diberikan auto dan negatif.
